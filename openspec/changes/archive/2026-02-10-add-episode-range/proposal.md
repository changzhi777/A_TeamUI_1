# add-episode-range 提案

## Why

当前项目卡片只显示项目名称、状态和基本信息，缺少对短剧项目非常重要的集数信息。对于短剧制作团队来说，能够快速了解项目的集数范围（如"第1-5集"）对于项目规划和进度跟踪非常重要。

## What Changes

### 新增功能

1. **集数范围字段** - 在项目数据模型中添加集数范围字段，支持自定义集数范围（如"第1-30集"）
2. **项目卡片显示** - 在项目卡片顶部（项目名称旁边）显示集数信息
3. **集数编辑功能** - 在项目创建和编辑时可以设置集数范围
4. **集数显示优化** - 使用醒目的样式显示集数，区别于其他项目信息

### 功能需求

- **数据模型**: 添加 `episodeRange` 字符串字段到 Project 接口
- **创建/编辑**: 在项目表单中添加集数范围输入框
- **卡片显示**: 在项目卡片顶部显示集数范围标签
- **样式设计**: 集数标签使用醒目的颜色和图标

### 技术影响

- 需要修改 `project-store.ts` 中的 Project 接口
- 需要修改 `project-form-dialog.tsx` 添加集数输入框
- 需要修改 `project-list.tsx` 中的 ProjectCard 组件显示集数
- 需要修改国际化文件添加集数相关翻译
- 需要在项目初始化时添加默认集数范围

### 验收标准

1. 用户创建项目时可以设置集数范围（如"第1-5集"）
2. 项目卡片在顶部显示集数范围标签
3. 集数标签使用醒目的颜色和样式
4. 集数信息可以在项目编辑时修改
5. 集数范围显示格式一致（如"第X-Y集"）
